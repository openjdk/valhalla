    exit

    public static void exit (int status)
    Initiates the shutdown sequence of the Terminates the currently running Java Virtual Machine. This method always blocks indefinitely. The argument serves as a status code; by convention, a nonzero status code indicates abnormal termination.

    This method calls the exit method in class Runtime . This method never returns normally.

    The call System.exit(n) is effectively equivalent to the call: 

     Runtime.getRuntime().exit(n)
    Parameters:
    status - exit status.
    Throws:
    SecurityException - if a security manager exists and its checkExit method doesn't allow exit with the specified status.
